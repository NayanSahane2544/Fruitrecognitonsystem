# 🍎 AI Agri-Tech Fruit Recognition System

A full-stack, machine-learning-powered web application designed to identify fruits from user-uploaded images and provide a comprehensive agricultural and nutritional analysis.

## ✨ Features
* **Custom CNN Deep Learning Model:** Accurately classifies 12 different types of fruits using a 13MB TensorFlow/Keras model.
* **Modern React Dashboard:** Built with Vite, featuring drag-and-drop uploads, UI skeleton loaders, and a responsive design.
* **Interactive Data Visualization:** Utilizes Recharts to display macronutrient breakdowns and Indian state cultivation statistics.
* **PDF Report Generation:** Users can export their AI analysis into a formatted PDF document.
* **Session History:** A built-in sidebar tracks recent scans with confidence scores, allowing users to clear their session data instantly.

## 🛠️ Tech Stack
* **Machine Learning:** TensorFlow, Keras, OpenCV, NumPy
* **Backend:** Python, Flask, Flask-CORS
* **Frontend:** React.js, Vite, Axios, Recharts, Lucide-React
